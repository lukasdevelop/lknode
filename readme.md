## Publishing to NPM

To start the package: https://github.com/lukasdevelop/lknode

```shell
$ npm install -g lknode
$ lknode start
$ npm install
$ npm run dev
```
# Commands 
```shell
$ lknode -h
$ lknode start
$ lknode g:c NameOfController
```
